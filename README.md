# 🌹 Un an avec toi

Un projet web minimaliste et interactif créé pour célébrer notre premier anniversaire.

## ✨ Fonctionnalités

* **Compteur de jours en temps réel** : Calcule automatiquement le temps écoulé depuis notre rencontre le 30 mars 2025.
* **Design Glassmorphism** : Une interface moderne, douce et épurée utilisant **Tailwind CSS**.
* **Interactivité "Love"** : Un bouton magique qui déclenche une pluie de cœurs animés (propulsé par **Lucide Icons**).
* **Optimisé pour mobile** : Parce que l'amour se consulte souvent entre deux cours ou dans le train.

## 🛠️ Stack Technique

* **HTML5 / CSS3** (Animations personnalisées)
* **Tailwind CSS** (Stylisation)
* **JavaScript Vanilla** (Logique du compteur et animations)
* **Lucide.dev** (Iconographie)
